<h2>Research-Assistant- AI Powered Text Summarizer Extension</h2>

<p>An intelligent browser extension that helps researchers, students, and professionals summarize selected text 
instantly using Google Gemini API.</p>

<p>Built with Java Spring Boot, this project demonstrates how AI can simplify research and enhance productivity 
through multilingual summarization and smart context understanding.</p>

<h3>Features</h3>

<p><b>Instant Summarization:</b> Select any text and get a concise, meaningful summary in seconds</p>
<p><b>Multilingual Support:</b> Works with multiple language, not just english</p>
<p><b>Browser Extension UI:</b> Easy to use interface that integrates directly with your browser</p>

<h3>Tech Stack</h3>
<p><b>User Interface:</b> HTML,CSS,JavaScript</p>
<p><b>Backend:</b> Java Spring Boot</p>
<p><b>AI API:</b> Google Gemini</p>
<p><b>AI API:</b>REST APIs, JSON</p>

<h3>How It Works</h3>
<p>1.The user highlights any paragraph or text on a web page</p>
<p>2.The browser extension sends the selected text to the Spring Boot backend</p>
<p>3.The backend communicates with Google Gemini API to generate a concise summary</p>
<p>4.The summarized text is displayed instantly within the extension UI</p>
